# Menu 菜单

按钮，用于开始一个操作

rain-ui 在 button 中添加了很多小细节，比如 hover 和 active 的颜色取自于由 primary-color 生成的色板，颜色和谐，再点击后还会有一个阴影的效果，不同类型的按钮有不同的效果

rain-ui 根据当前色彩计算设置暗色主题之后的颜色，以及紧凑模式，点击右下角试试叭~
